# Spring Security JWT


Sample code to present login and register  with JWT (JSON Web Token) and document it with Swagger v2 (also supporting JWT)
and simple CRUD.

## Technologies
Project is created with:
* JAVA version 8
* Spring Boot
* Spring Data
* Spring Security
* H2 database
* Maven
* Lombok
* Swagger
